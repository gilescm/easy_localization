## Forker's Notes
Forked so that I could add a `preloaderWidget` parameter to replace the `preloaderColor`. The `preloaderWidget` was in the dev/master branch of the [Easy localization repo](https://github.com/aissat/easy_localization) but hadn't been put to stable release yet. `preloaderWidget` was added officially in 2.2.2 so this fork is now redundant.
